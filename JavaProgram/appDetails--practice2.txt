Description




Olivia, an app developer, wants to track her apps' ratings on the Play Store to improve their features. The manager asks for your help in developing a Java program to meet this requirement.

Functional Requirement:

Req. #

Requirement Description  

Method Name

Parameters

Responsibilities

1

Add the appName and its rating into appMap.

 addAppDetails

String appName, double rating  

This method should add the appName and its rating into the appMap, which is implemented as a HashMap.

Note: The appMap takes appName as key and rating as value.

Constraints:

appName is case-sensitive.
2

Find rating of the given app.



 findAppRating

String appName

This method accepts appName as the argument and find the rating based on the specified appName.

 Constraints:

appName is case- sensitive.
The appRating should be returned as double.
When the appName is not present in the Map, return -1.


You are provided with the main method in the UserInterface class as code template, and it is excluded from evaluation.

Note:

Edit only the AppInfo class to implement the business requirements.
The methods should be public, and the attributes of the class should be private.

Ensure that the names for classes, attributes, and methods are provided as specified in the question description.
Please do not use System.exit(0); to terminate the program.
