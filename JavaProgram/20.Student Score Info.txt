Description

Objective:

To work with streams

Concept Explanation:

Java streams offer a functional approach to process collections, arrays, or input/output operations. With operations like filtering, mapping, and reducing, streams enable concise and efficient data manipulation, promoting declarative programming for enhanced code readability and maintainability.

Concept Implementation:

The Student class manages student details such as ID, name, and percentage score, while the StudentUtility class handles a list of students. It converts the list to a stream and filters students based on a provided score, returning a sorted list. The UserInterface class manages input for student data and score, utilizing StudentUtility methods for filtering and displaying results.

Scenario:

University of Toronto organizes a carrier fair for final semester students every year. They wish to group and filter the student info based on the percentage score of the students. Develop a Java application for the same using Streams.
