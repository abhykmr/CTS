Description


Objective:

To work with interface, static method.

Concept Explanation:

Interfaces in Java define a contract for classes, specifying method signatures that implementing classes must provide. Static methods in interfaces are shared among all implementing classes and can be called without creating instances.

Concept Implementation:

The FitnessMonitor interface provides a contract for finding fitness levels, including a static method for calculating burnt calories. The UserFitnessData class implements this interface, storing user fitness data and calculating fitness levels. The UserInterface class interacts with users to input data and displays fitness information.

Scenario:

ABC Fitness, a leading fitness center, wants to create a software tool to help their clients track their daily fitness activities. The tool should have the capability to monitor metrics such as calories burnt, steps taken, distance covered, and other relevant data. It should also provide insights and recommendations based on the clients' fitness goals and progress. As a software developer, you have been tasked with creating the software tool to meet ABC Fitness's requirements.

 Requirement 1: Calculate the fitness level of the user.

Type (Class)

public String findFitnessLevel()

This method finds the fitness level based on the fitness score, the fitness score is calculated based on stepsTaken, distanceCovered and weight of the user and returns the fitness level based on the calculated fitness.

Refer the formula below for the fitness level calculation.

 

Formula to calculate the fitness level of the user:

FitnessScore = [(0.1 x stepsTaken) + (1.8 x distanceCovered)] / weight in kilograms

 Requirement 2: Calculate the calories burnt per day.

Type (Class)

Methods

Responsibilities

FitnessMonitor

public static double calculateCaloriesBurnt(double weight, double distanceCovered, boolean runningStatus)

This method calculates the calories burnt by the user based on the weight, distanceCovered, and runningStatus then, returns the calories burnt as a result.

If runningStatus is true:

Calories Burnt = 0.75 * weight * distanceCovered

If runningStatus is false:

Calories Burnt = 0.53 * weight * distanceCovered

 

Note: runningStatus is a boolean variable that specifies whether the user was running(true) or walking(false) during their fitness routine.
