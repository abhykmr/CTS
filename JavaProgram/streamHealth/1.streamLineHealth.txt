Concept Explanation:  

In Java, data types specify the type of values that variables can hold, such as int for integers, double for floating-point numbers, char for characters, and boolean for true/false values. These data types define the size and type of data that can be stored in variables, ensuring proper memory allocation and manipulation. 

Concept Implementation: 

Data types are used to store user input for weight and height. In this case, the double data type is used to accurately represent decimal values for weight and height measurements.
