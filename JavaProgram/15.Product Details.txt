Objective:

To work with user defined exception.


Concept Explanation:

Java allows for the creation of custom exceptions, which are subclasses of the Exception class. Custom exceptions tailor error handling to specific needs, involving the creation of bespoke exception classes and triggering them using the 'throw' keyword.

Concept Implementation:
Here, custom exception usage in Java for precise error handling. It validates product prices and updates stock based on user input. A custom exception class, InvalidPriceException, handles invalid product prices. In the main method, user input for product details undergoes iterative processing, validating prices with the validateProductPrice method. Exception handling in main catches InvalidPriceException, ensuring smooth handling of invalid inputs.
